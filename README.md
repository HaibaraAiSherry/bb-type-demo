### 通讯

`ui` 层通过 emit 与 端层进行通讯，以此来拿到「平台级 API」能力，或是其他等功能的调用。

### 痛点及解决方案

- 不同端会有不同的布局

  - 借鉴 taro 的开发方式，index.平台.tsx，根据指定平台来编译、运行。

- 不同端会有不同的功能

  - 涉及端能力

    - 布局统一触发的话，则与 `端` 进行通讯，以完成此功能

  - 不涉及端能力，只需因不同端而做不同处理

    - 将不同的部分，拆成 `action 层`
